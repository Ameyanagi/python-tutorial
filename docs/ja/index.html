# Python チュートリアル {.unnumbered}

包括的な Python チュートリアルへようこそ！この本では Python の基礎から応用まで学習できます。

## 学習内容

このチュートリアルは様々なスキルレベルに対応し、以下の内容をカバーします：

- **環境構築** - uv、poetry、miniforge、Git/GitHub、VS Code
- **Python の基本** - 構文、データ型、制御フロー、関数
- **オブジェクト指向プログラミング** - クラス、継承、高度なパターン
- **高度なトピック** - 型ヒント、非同期プログラミング、マルチプロセシング
- **実際の応用** - データサイエンス、自動化、Web 開発

## チュートリアルの特徴

- 🖥️ **インタラクティブな例** - 実行結果付きのコード
- 📝 **演習とクイズ** - 理解度をテスト
- 🚀 **実践プロジェクト** - 実際のアプリケーションを構築
- 🎯 **複数の形式** - 書籍、スライド、PDF
- 🌐 **バイリンガル** - 英語と日本語で利用可能

## 前提条件

- 基本的なコンピュータリテラシー
- 学習と実験への意欲
- Python 3.12+（インストール方法も説明します！）

## このチュートリアルの使い方

1. **順次学習** - 最良の結果を得るために章順に進む
2. **コード実践** - すべての例を自分で実行する
3. **演習完了** - 学習を強化する
4. **プロジェクト構築** - 実際の問題に知識を適用する
